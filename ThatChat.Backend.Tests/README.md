# ThatChat.Backend.Tests

Тесты для [`ThatChat.Backend`](../ThatChat.Backend)

Запускать как обычно

При запуске тестов, в корневую папку генерируется `allure-results`

```bash
dotnet test
```

[CI/CD](../.github/workflows/cicd.yaml) запускает данные тесты
и отгружает на https://chanan-duy.github.io/ThatChat.Backend
